# Web Demo

A web demo using Bootstrap 5, jQuery and ECharts.

## Pages

| Page | Description |
|------|----------|
| `index.html` | Homepage with 2 buttons |
| `instruction.html` | User guide for Python scripts |
| `visualize.html` | Data table visualization with DataTables |

## How to Run

Open `index.html` in your browser:

```bash
xdg-open index.html
# or
firefox index.html
```

## Tech Stack

- Bootstrap 5.3.3
- jQuery 3.7.1
- DataTables 2.1.8
- Bootstrap Icons 1.11.3

All loaded from CDN, no installation required.
