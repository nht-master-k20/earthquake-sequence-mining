Danh sách Features đề xuất:
Features cơ bản (đã có):
mag - Độ lớn động đất
place - Vị trí mô tả
time - Thời gian xảy ra (timestamp)
latitude - Vĩ độ
longitude - Kinh độ
depth - Độ sâu (km)
Features bổ sung (nên thêm):
magType - Loại magnitude ("mw", "mb", "ml", "ms")

Quan trọng vì các loại magnitude khác nhau có ý nghĩa khác nhau
status - Trạng thái đánh giá ("reviewed", "automatic", "preliminary")

Cho biết độ tin cậy của dữ liệu
tsunami - Cảnh báo sóng thần (0 hoặc 1)

Feature quan trọng cho đánh giá rủi ro
sig - Significance score (385)

Chỉ số quan trọng của sự kiện (0-1000+)
rms - Root Mean Square error (0.26)

Độ chính xác của vị trí xác định
net - Network code ("ak", "us", etc.)

Nguồn dữ liệu
type - Loại sự kiện ("earthquake", "quarry blast", etc.)

Phân biệt động đất tự nhiên vs nhân tạo
id - Event ID ("ak99515ijv2y")

Mã định danh duy nhất
Features bổ sung nếu cần thêm:
gap - Azimuthal gap (độ)

Khoảng trống trong mạng lưới quan sát
dmin - Minimum distance (độ)

Khoảng cách đến trạm quan sát gần nhất
nst - Number of stations

Số trạm được sử dụng trong tính toán
felt - Số người báo cáo cảm nhận

Thông tin về tác động
cdi - Community Decimal Intensity

Cường độ dựa trên báo cáo cộng đồng
mmi - Modified Mercalli Intensity
