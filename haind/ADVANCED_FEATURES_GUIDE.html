2. LSTM Temporal Features

7 Features
Tên Feature Kiểu Ý Nghĩa LSTM Công Thức
time_since_last_event Float ⚡ CRITICAL: Thời gian kể từ event trước
Δt = current_time - prev_time
time_since_last_M5 Float Thời gian kể từ động đất M≥5.0 gần nhất Big earthquake cycle
interval_lag1 Float ⚡ Sequence pattern: Interval từ event t-1 Δt[t-1]
interval_lag2 Float ⚡ Sequence pattern: Interval từ event t-2 Δt[t-2]
interval_lag3 Float ⚡ Sequence pattern: Interval từ event t-3 Δt[t-3]
interval_lag4 Float ⚡ Sequence pattern: Interval từ event t-4 Δt[t-4]
interval_lag5 Float ⚡ Sequence pattern: Interval từ event t-5 Δt[t-5]

🧠 Tại sao LSTM cần Temporal Features?

Time-series prediction requires understanding temporal patterns. Interval sequences help LSTM learn the "rhythm" of earthquake occurrence. Short intervals → high activity period. Long intervals → stress accumulation phase.

3. LSTM Rolling Statistics Features

9 Features
Tên Feature Kiểu Ý Nghĩa LSTM Window
rolling_count_1h Integer Số events trong 1 giờ gần nhất 1 hour
rolling_count_24h Integer Số events trong 24 giờ gần nhất 24 hours
rolling_count_7d Integer Số events trong 7 ngày gần nhất 7 days
rolling_mean_mag_1h Float Mean magnitude trong 1 giờ 1 hour
rolling_mean_mag_24h Float Mean magnitude trong 24 giờ 24 hours
rolling_mean_mag_7d Float Mean magnitude trong 7 ngày 7 days
rolling_max_mag_1h Float Max magnitude trong 1 giờ 1 hour
rolling_max_mag_24h Float Max magnitude trong 24 giờ 24 hours
rolling_max_mag_7d Float Max magnitude trong 7 ngày 7 days

🧠 Tại sao LSTM cần Rolling Features?

Multi-time-scale analysis helps LSTM capture both short-term and long-term patterns. 1h = immediate activity, 24h = daily pattern, 7d = weekly trend. Mean mag = energy level, Max mag = largest recent event.

4. Target Variables

3 Features
Tên Feature Kiểu Ý Nghĩa - Prediction Task Range
target_time_to_next Float 🎯 TARGET 1: Thời gian đến earthquake tiếp theo (giây) 0 - ∞

🎯 Multi-Task Learning

Dual prediction: Model dự đoán CẢ thời gian VÀ độ lớn của earthquake tiếp theo. Binary target là optional cho classification task (động đất nguy hiểm hay không).
