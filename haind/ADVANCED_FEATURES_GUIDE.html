Số lượng feature: 11 feature nâng cao

Dữ liệu: Earthquake catalog (M ≥ 3.0)

Chú thích: THỜI GIAN ĐỘ LỚN CẢ HAI GIÁN TIẾP

1. FEATURE PHÁT HIỆN DƯ CHẤN (AFTERSHOCK DETECTION)

Tên Feature Ý nghĩa Dự đoán Cơ sở khoa học / Công thức Ví dụ
is_aftershock Event có phải dư chấn không (Gardner-Knopoff algorithm) CẢ HAI
Time: t < 10^(0.5×M - 1.0) giây
Dist: d < 10^(0.123×M + 0.034) km
 Mainshock M7.0:
- Window: 316 ngày, 42 km
- Event trong window → dư chấn
→ Event tiếp theo sớm hơn, nhỏ hơn
mainshock_id ID của trận chính gây ra dư chấn này GIÁN TIẾP Liên kết chuỗi dư chấn với trận chính Dư chấn #5 của mainshock #1234
→ Model học được: "đây là dư chẫn muộn"
→ Event tiếp theo càng muộn
mainshock_mag Độ lớn của trận chính CẢ HAI Båth's Law: M_aftershock ≈ M_main - 1.2 M_main = 7.0
→ Aftershocks: ~5.8-6.0
→ Chuỗi dư chấn kéo dài hàng năm
→ Event tiếp theo ≤ 7.0

2. FEATURE KHOẢNG CÁCH ĐỨT GÃY (FAULT PROXIMITY)

Tên Feature Ý nghĩa Dự đoán Cơ sở khoa học / Công thức Ví dụ
dist_to_5th_neighbor_km Khoảng cách đến event thứ 5 gần nhất THỜI GIAN KD-tree spatial query, Haversine distance 5 km: Gần fault hoạt động
→ Seismicity cao → Event sớm
100 km: Xa fault → Event muộn
dist_to_20th_neighbor_km Khoảng cách đến event thứ 20 gần nhất ĐỘ LỚN Proxy cho fault maturity Cô lập (200 km): Fault bị khóa
→ Đang tích tụ stress → Có thể M lớn
Dày (10 km): Fault đang creep → M nhỏ
seismicity_density_100km Số event trong bán kính 100 km CẢ HAI KD-tree count within radius High (>50): Fault đang creep
→ Sớm, nhỏ
Low (<10): Fault bị khóa
→ Muộn, có thể lớn

3. FEATURE ỨNG SUẤT COULOMB

Tên Feature Ý nghĩa Dự đoán Cơ sở khoa học / Công thức Ví dụ
coulomb_stress_proxy Ứng suất tích tụ từ các event trước (đơn giản hóa) THỜI GIAN
stress = Σ(M² / d) × scale
Coulomb Stress Transfer (phiên bản proxy) 		High stress: Gần ngưỡng failure
→ Event tiếp theo rất sớm
Low stress: Đang tích tụ
→ Event muộn hơn

4. FEATURE ĐỘNG ĐẤT KHU VỰC (REGIONAL)

Tên Feature Ý nghĩa Dự đoán Cơ sở khoa học / Công thức Ví dụ
regional_b_value Hệ số b Gutenberg-Richter (200 km, 5 năm) ĐỘ LỚN
log₁₀(N) = a - b×M
b = slope của frequency-magnitude 		b = 0.6 (thấp): Stress cao
→ Ít event nhưng lớn
b = 1.5 (cao): Stress thấp
→ Nhiều event nhưng nhỏ
seismic_gap_days Ngày kể từ M5+ gần nhất (200 km) CẢ HAI Seismic Gap Hypothesis < 100 ngày: Vừa có M5+
→ Sớm, có thể dư chấn
> 1000 ngày: Gap dài
→ Có thể lớn (nếu fault locked)
regional_max_mag_5yr M lớn nhất trong khu vực (5 năm) ĐỘ LỚN Historical seismicity potential M_max = 6.5: Vùng có thể M6.5+
→ Upper bound cho event tiếp theo
→ Event ≤ 6.5 (thường)
